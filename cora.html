Welcome to the IMDB Profitability Analysis Page

These page illustrates the results from trying to predict the profitability of a movie based on whether it is English language movie or not.

Three machine learning models were used in this analysis: linear regression, random forest regression, and support vector machine regressor. The R squared calculation was used as a benchmark to compare all three.

Linear Regression

For the linear regression the R squared was 0.0199. This is a scatter plot of the linear regression.

Random Forest Regression

For the Random Forest Regression the R squared was 0.0199. This is a seaborn residual plot of the Random Forest Regression.

Support Vector Machine Regression

For the Support Vector Machine Regression the R squared was -0.0257. This is a seaborn residual plot of the Support Vector Machine Regression.
