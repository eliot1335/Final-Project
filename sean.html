Welcome to IMDB Profitability Analysis Page

These page illustrates the results from trying to predict the profitability of a movie based on the genre and the description.

Linear Regression

Linear Regression had a Training Data Accuracy of 0.9987 and a Testing Data Accuracy of 0.2820.

This is a confusion matrix for the Linear Regression run. There should be a solid line running from the lower left to the upper right. This plot does not show this. This illustrates the data did not converge well.

This is the penalty regression for the Linear Regression run. It shows that a small penalty actually increases performance by about 5%. As the penalty approaches 1, we are back down to the original accuracy.

Keras Neural Net

Keras Neural Net had a Training Data Accuracy of 0.9982 and a Testing Data Accuracy of 0.2673. I tried multiple iterations on the architecture to improve this score without avail. I believe there is something fundamental that I am missing.

These are plots that illustrates the training and validation loss and accuracy. We can see the training data quickly goes to 100% accuracy and 0 loss. In contrast the validation data goes from 35% accuracy down to 27% accuracy as the training progresses. I could not find an answer to this

Random Forest

Random Forest had a baseline accuracy of 0.3643 and after hyper-parameter tuning it is .3669. This is the best model out of the three and I spent more time doing analysis of it.

This picture illustrates the tree that is generated from the best model. It is 77 levels deep.

This is a regression of the feature size. In the baseline model there are 2028 features. We see we can reduce the feature size by half and not have a significant impact on the accuracy.

This is a regression of the sample size. We see that the sample size is not affecting the accuracy.

This is a regression of the tree size. We see even forcing the tree size to be 30 deep does not have a major effect on the accuracy.
